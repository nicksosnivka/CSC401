Overview

This animation represents double hashing. The first hash function is to take the key mod the table size (H1 = key mod 16). The second hash function is H2 = 2 (key mod 4) + 1. Combined together, to insert, the slot is determined by (H1 + i H2) mod 16. If the hash function yields a collision, the number of collisions increases by 1 and the hash function will run. If there is another collision, collisions will increment with one and the hash function will run again and again until an empty cell is found. The collided cells will have the resident value shown in red. The newly inserted element is shown in green.
